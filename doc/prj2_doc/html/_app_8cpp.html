Solves the Traveling Salesman Problem (TSP) using the Held-Karp algorithm.

Parameters
graphThe graph representing the TSP problem.
bestPathThe best path found by the algorithm.
bestCostThe cost of the best path found.

This function applies the Held-Karp algorithm to find the optimal solution for the TSP. @timecomplexity The time complexity of this function is O(2^n * n^2), where n is the number of vertices in the graph.

Remarks
Pros: Guarantees the optimal solution for the TSP. Suitable for small to medium-sized graphs. Cons: High time complexity, impractical for large graphs due to exponential growth.
